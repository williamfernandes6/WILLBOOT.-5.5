# 🧠 WillBoot PRO - Frontend

Este é o pacote completo do frontend do projeto **WillBoot PRO**.
Pronto para ser publicado em `https://willboot.github.io`.

---

## 📁 Estrutura dos Arquivos:

- `index.html`: Tela principal com logo, frase e botões de acesso
- `planos.html`: Seleção de planos VIP + campo de voucher
- `sinais.html`: Painel com IA real (X estimado, tempo, confiança)
- `style.css`: Estilo escuro com detalhes em vermelho 🔥
- `script.js`: Integração com o backend (Render)
- `sinal-novo.mp3`: Som de alerta "Sinal novo!"
- `logo.png`: Logo da tua marca (substituível)

---

## 🚀 Como Publicar no GitHub Pages

1. Acesse [github.com/new](https://github.com/new)
2. Crie um novo repositório com o nome exato:
   **`willboot.github.io`**
3. Faça upload de todos os arquivos desse ZIP para esse repositório:
   - Clique em **"Add files" → "Upload files"**
   - Arraste todos os arquivos
   - Clique em **"Commit changes"**
4. Pronto! Acesse o site em:
   ```
   https://willboot.github.io
   ```
